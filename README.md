# Selfstabilization
W pliku _Main.java_ znajduje się zmienna _N_ określająca ilość procecsów biorących udział oraz tablica _processes_states_ określająca początkowe stany poszczególnych procesów

Program po uruchomieniu wyświetla aktualne decyzje podejmowane przez proces a na końcu informuje o zakończeniu i pokazuje aktualne stany procesów
